# distantjetlines' Portal 1 Test Chamber Texture Collection


A collection of PBR enabled Portal 1 testchamber textures for Portal 2: Community Edition.. All the basics you should need.

Provided in 1024px/128u resolution.

Credit required if used.

### Textures included:
* Concrete walls in 2 variations
* BTS styled concrete wall
* 64u square testchamber wall
* Concrete floor and ceiling
* Metal wall in one size for resizing
* Metal floor in two color variations
* 16u plastic column
* 8u hazard/panel edge

Other textures (white plastic trim and observation textures) available at https://github.com/distantjetlines/djtextures_clean

Technical details: RGBA8888 w/ ZSTD lvl22. Many textures have parallax heightmaps embedded, but parallax disabled for performance reasons and personal visual preferences. 

For those textures, VMT files have parallax feautures commented out. If you choose to enable them, accept that you might have to tweak those settings to achieve a desirable result.
